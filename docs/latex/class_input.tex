\doxysubsection{Detailed Description}
Static input system wrapper around GLFW. 

Provides keyboard state polling and mouse delta tracking. Designed as a global input manager for the engine.

Handles:\+
\begin{DoxyItemize}
\item Keyboard input (WASD, actions, etc.)
\item Mouse movement (delta-\/based FPS camera control)
\item Cursor mode switching (normal /\+ locked) 
\end{DoxyItemize}
