\doxysubsection{Detailed Description}
Responsible for rendering a scene using a camera and shader pipeline. 

The \doxylink{class_renderer}{Renderer} is the core of the rendering system. It takes a \doxylink{class_scene}{Scene} and a \doxylink{class_camera}{Camera} and produces a final image by:\+
\begin{DoxyItemize}
\item binding shaders
\item setting uniforms (view/\+projection matrices)
\item issuing draw calls for all meshes in the scene 
\end{DoxyItemize}
