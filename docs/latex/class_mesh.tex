\doxysubsection{Detailed Description}
Represents a renderable 3D mesh stored on the GPU. 

A mesh encapsulates:\+
\begin{DoxyItemize}
\item Vertex data (VBO)
\item Index data (EBO)
\item Vertex array configuration (VAO)
\end{DoxyItemize}

It is responsible for storing geometry in GPU memory and issuing draw calls.
