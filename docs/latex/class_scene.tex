\doxysubsection{Detailed Description}
Represents a collection of renderable meshes. 

The \doxylink{class_scene}{Scene} class owns and manages a set of \doxylink{class_mesh}{Mesh} objects. Ownership is exclusive and handled via std::\+unique\+\_\+ptr.
