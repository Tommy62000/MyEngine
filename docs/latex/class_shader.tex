\doxysubsection{Detailed Description}
Encapsulates an Open\+GL shader program. 

The \doxylink{class_shader}{Shader} class handles loading, compiling, and linking vertex and fragment shaders into a single GPU program. It also provides utility functions for setting uniform values.
