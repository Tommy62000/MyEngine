Detailed Description

Represents a collection of renderable meshes.

The Scene class owns and manages a set of Mesh objects. Ownership is exclusive and handled via std::unique_ptr.

void Scene::addMesh ( std::unique_ptr< Mesh > mesh)

Adds a mesh to the scene.

Transfers ownership of the given mesh to the Scene. After the call, the caller no longer owns the mesh.

Parameters
meshUnique pointer to the Mesh to add. Must be passed with std::move.
Note
The input pointer becomes null after the call.

const std::vector< std::unique_ptr< Mesh > > & Scene::getMeshes ( ) const

Retrieves all meshes in the scene.

Provides read-only access to the internal container. The structure of the container cannot be modified, but the Mesh objects themselves may still be mutable.

Returns
Constant reference to the vector of meshes.
Warning
Modifying the returned Mesh objects may affect the Scene state.
