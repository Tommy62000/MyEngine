Detailed Description

Static input system wrapper around GLFW.

Provides keyboard state polling and mouse delta tracking. Designed as a global input manager for the engine.

Handles:

  • Keyboard input (WASD, actions, etc.)
  • Mouse movement (delta-based FPS camera control)
  • Cursor mode switching (normal / locked)
