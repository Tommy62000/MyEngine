Detailed Description

Represents a renderable 3D mesh stored on the GPU.

A mesh encapsulates:

  • Vertex data (VBO)
  • Index data (EBO)
  • Vertex array configuration (VAO)

It is responsible for storing geometry in GPU memory and issuing draw calls.

glm::mat4 Mesh::getModelMatrix ( ) const

Computes the model transformation matrix.

Combines translation, rotation, and scaling into a single matrix that transforms object space into world space.

Order of transformations:

  1. Scale
  2. Rotate (Z -> Y -> X or implementation-defined)
  3. Translate
