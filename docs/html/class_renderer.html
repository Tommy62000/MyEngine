Detailed Description

Responsible for rendering a scene using a camera and shader pipeline.

The Renderer is the core of the rendering system. It takes a Scene and a Camera and produces a final image by:

  • binding shaders
  • setting uniforms (view/projection matrices)
  • issuing draw calls for all meshes in the scene

void Renderer::draw ( const Scene & scene,
const Camera & camera )

Renders a complete scene from the perspective of a camera.

This function performs the full rendering pipeline:

  1. Activates the shader program
  2. Uploads camera view and projection matrices
  3. Iterates over all scene objects
  4. Applies model transformations
  5. Issues draw calls for each mesh
Parameters
sceneScene containing all renderable objects
cameraCamera defining view and projection matrices
